#Gestion Tools

Biblioteca con métodos comunes de tratamiento de archivos, streams, conexiones a base de datos (JDBC), manejo de fechas, etc.

**Versión 0.0.3**
  
* Se agregan a la clase DateTools métodos para obtener el número de semana del año, a partir de una fecha dada.

* Se agrega un método para obtener un Date que representa al día de la semana (Lunes, Martes, Miercoles, etc...) de acuerdo al año y número de semana proporcionados.

* Se agrega clase para representar un periodo (fecha inicio y fecha fin), como complemento del método DateTools.getDiaDeSemanaDate.

* Se agrega clase para obtener parámetros JNDI del servidor.
